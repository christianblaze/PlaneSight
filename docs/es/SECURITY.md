# Seguridad

## Software Experimental

PlaneSight es un proyecto experimental de investigación.

El software no ha sido sometido a una auditoría profesional de seguridad y no debe considerarse seguro para almacenar Bitcoin, claves privadas, contraseñas, comunicaciones sensibles o secretos valiosos.

---

## No Almacene Cantidades Significativas de Bitcoin

PlaneSight fue inspirado originalmente por el concepto de incrustar claves privadas de Bitcoin dentro de medios digitales.

Este concepto sigue siendo un área de investigación.

Los usuarios deben asumir que la información oculta puede:

- Corromperse
- Volverse irrecuperable
- Ser detectada
- Ser parcialmente destruida
- Ser eliminada por procesamiento de imagen, audio o video

No utilice PlaneSight para almacenar cantidades significativas de Bitcoin.

---

## Riesgos Conocidos

Los riesgos potenciales incluyen:

- Daño por compresión JPEG
- Redimensionamiento de imágenes
- Transcodificación de video
- Recompresión de audio
- Eliminación de metadatos
- Conversión de formatos
- Errores de software
- Error humano
- Detección mediante estegoanálisis

Las versiones futuras podrían mejorar la resiliencia, pero no se ofrece ninguna garantía.

---

## Modelo de Amenazas

PlaneSight actualmente se enfoca en:

- Ocultamiento experimental de información
- Investigación de resiliencia frente a transformaciones
- Investigación de codificación multicapa

PlaneSight actualmente no proporciona:

- Secrecía garantizada
- Recuperación garantizada
- Almacenamiento resistente a manipulaciones
- Custodia segura de Bitcoin

---

## Uso Responsable

PlaneSight debe utilizarse para:

- Investigación
- Educación
- Experimentación
- Desarrollo de pruebas de concepto

PlaneSight no debe utilizarse como reemplazo de soluciones criptográficas establecidas para almacenamiento seguro.

---

## Reporte de Problemas de Seguridad

Si descubre un problema de seguridad, error o vulnerabilidad, abra un GitHub Issue describiendo:

- El método afectado
- Pasos para reproducir el problema
- Comportamiento esperado
- Comportamiento observado

Las contribuciones que mejoren la fiabilidad, resiliencia y seguridad son bienvenidas.
