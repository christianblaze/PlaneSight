# Experimental v0.9.1

## Added

- PNG Stealth mode
- JPEG Resilient mode
- Resize Lab
- Scale Lab
- DCT Lab
- Wavelet Lab
- Watermark Lab
- Metadata embedding
- Multi-method layering
- Compatibility verification
- Method interference detection
- Payload capacity analysis
- Layer ordering and recovery verification

## Changed

- Expanded PlaneSight from a simple steganography proof-of-concept into a multi-method steganography research platform.
- Added experimental resilience testing against JPEG conversion, resizing, scaling, and image transformations.
- Added support for combining multiple encoding methods simultaneously.
- Improved payload verification and decoding reliability.
- Improved user interface and research tooling.

## Notes

This release represents the current experimental research platform and establishes the foundation for future image, audio, video, metadata, and multi-layer cryptographic embedding research.

# Original Proof-of-Concept

## Added

- Basic image steganography
- Text embedding
- Text extraction
- Error correction experiments
- JPEG survivability experiments

## Notes

The original PlaneSight proof-of-concept was created to explore the idea of embedding Bitcoin private keys into digital images.

The broader vision was to investigate whether digital media could carry embedded Bitcoin value, potentially creating new forms of digital ownership, preservation incentives, and media-linked value systems.
