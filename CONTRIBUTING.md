# Contributing

Thank you for your interest in PlaneSight.

This project welcomes:

- Bug reports
- Feature requests
- Pull requests
- Research ideas
- Steganography experiments

## Development Philosophy

PlaneSight is an experimental research platform.

Contributors are encouraged to:

- Prioritize reproducible experiments
- Document tradeoffs
- Provide test images where possible
- Explain resilience assumptions

## Areas of Interest

- Image steganography
- Audio steganography
- Video steganography
- Error correction
- JPEG resilience
- Resize resilience
- Frequency-domain encoding
- Bitcoin-related applications

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Submit a pull request

Thank you for helping advance steganography research.
